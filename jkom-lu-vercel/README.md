## Quick start

1. Install dependencies:
   ```
   npm install
   ```
2. Run dev:
   ```
   npm run dev
   ```
3. Add environment variables (Vercel or .env.local):
   ```
   NEXT_PUBLIC_PAYPAL_CLIENT_ID=your-paypal-client-id
   NEXT_PUBLIC_PAYPAL_PLAN_ID=your-paypal-plan-id
   ```
4. Deploy to Vercel.
